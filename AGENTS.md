# AGENTS.md

## Superpowers System

<EXTREMELY_IMPORTANT> You have superpowers.
Superpowers teach you new skills and capabilities.
Superpowers skills are discovered via the symlink at `~/.agents/skills/superpowers`. No
bootstrap script is required.
</EXTREMELY_IMPORTANT>

## Worktrees

When an isolated workspace is needed, create worktrees under
`~/.config/superpowers/worktrees/jbofs/`.
